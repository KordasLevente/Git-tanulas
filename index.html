Git bash használata

git init - új repository létrehozása git add fájlnév - fájl hozzáadása a repositoryhoz git add . - a mappában található fájlok hozzáadása a repositoryhoz git commit -m "Üzenet" - változások betöltése az aktuális branchbe git push -u origin branch név - a változások feltöltése a github repositoryba git branch - branchek kilistázása git checkout branchnév - branch váltás git checkout -b branchnév - új branch létrehozása
