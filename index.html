Parancs Mit csinál

git commit -m "Üzenet" stage feltöltése az aktuális branchbe
git push -u origin branch név a változások feltöltése a remote repositoryba
git branch branchek kilistázása
git checkout branchnév branch váltás
git branch branchnév
git checkout -b branchnév		 új branch létrehozása
