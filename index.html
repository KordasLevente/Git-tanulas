Git bash használata

Parancs Mit csinál Megjegyzés
git init üres helyi repository létrehozása
git clone link remote repository lekérése
git add fájlnév fájl hozzáadása a stagehez Ha a fájlnév helyett . van akkor a repoban lévő összes fájlt hozzáadja a stagehez
git commit -m "Üzenet" stage feltöltése az aktuális branchbe
git push origin branch név a változások feltöltése a remote repositoryba -u-val alapértelmezetté tehetünk egy repot és akkor nem kell beírni az elérést
git branch branchek kilistázása
git checkout branchnév branch váltás
git branch branchnév új branch létrehozása Alternatíva:
git checkout -b branchnév
git pull repository név lekérés a remote repositoryból --allow-unrelated-histories
